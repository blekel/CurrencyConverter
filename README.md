# Currency Converter

Sample Android app for real-time currency conversion.

## 📱 Features
- 🔄 Currency conversion based on the current exchange rate
- 🌍 Support for multiple currencies
- 📡 Automatic data update
- 🎨 Modern and user-friendly interface

## 🛠️ Technologies
- MVVM (Model-View-ViewModel) architecture for separating logic and UI
- Kotlin
- Jetpack Compose for UI
- Retrofit for working with API
- Coroutines Flow for asynchronous data handling
- Koin for dependency injection
- JUnit and UI tests
